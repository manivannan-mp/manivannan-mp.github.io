A template you can copy for every new article. Delete this once your first real post is live.

Writing structure

Use normal HTML tags — they're already styled:

  • <h2> for section headings
  • <p> for paragraphs
  • <blockquote> for pull quotes
  • <pre><code> for code blocks
Ship small, write it down, repeat. The best portfolio is a trail of things you actually built.
