This is a starter article. To publish your own, duplicate this file
(`content/posts/example-post.mdx`), rename it to something like
`content/posts/my-first-post.mdx`, and replace this content. Remove
`draft: true` from the frontmatter and it appears on the writing index.

## Writing structure

Write in Markdown — everything is already styled:

- `##` for section headings
- Regular paragraphs for body text
- `>` for pull quotes
- Fenced code blocks for code

> Ship small, write it down, repeat. The best portfolio is a trail of things you actually built.

## A code block

```js
function ship(idea) {
  const prototype = build(idea);
  return prototype.iterate();
}
```
